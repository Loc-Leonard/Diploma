# Нарушение (инспектор контрольного органа) + предписание
*Зачем*: инспектор фиксирует нарушения нормативки по классификатору контрольного органа, прикрепляет фото/доки, фиксируются время+место; зачем рассматривает исправления. Также он может инициировать лабораторный отбор проб материала

*Что хранить*:
- `id`, `object_id`
- Автор (инспектор)
- Код классификатора нарушений (контрольного органа)
- Описание
- Статус:
	- `OPEN`
	- `FIX_SUBMITTED`
	- `ACCEPTED / REJECTED`
- Гео + время фиксации
- Вложения (фото/доки)
- Предписание:
	- `violation_id`
	- Требования/срок устранения
	- Текст предписания
- Отдельная сущность LabSampleRequest:
	- `id`, `object_id`, `delivery_id`
	- Кто инициировал
	- Причина/комментарий
	- Статус (создано/в работе/закрыто)
	- Дата/время/гео
