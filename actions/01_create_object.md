# Создать или активировать объект (стройконтроль)
**Цель**: перевести объект из "планируемых" в "в работе" и формально зафиксировать старт
*Важно*:
- объект есть в списке планируемых
- наступила плановая дата начала работ

*Сценарий*:
1. Просмотр списка карточек объектов, на каждой видно полигон, состав работ, сетевой график
2. В дату начала работ стройконтроль:
	- активирует объект
	- назначает прораба со стороны подрядчика
3. Заполнят чек-лист акта открытия (инициализация объекта)
4. Прикрепляет акт открытия (файл) к карточке объекта.
5. Переводит объект в статус ACTIVE ("в работе")

**Результат**:
- у прораба появляется объект в списке доступных
- по объекту становятся доступны журнал работ, материалов, замечаний
