# Введение графика и состава работ (стройконтроль прораб)
### 1. Просмотр графика (все участники по назначению)
- Пользователь открывает объект -> видит состав работ и график (Гант)

### 2. Корректировка графика (прораб предлагает -> стройконтроль утверждает)
*Прораб*:
- Вносит предложение изменения плановых дат/объемов
- Система фиксирует "изменение требует согласнования"

*Стройконтроль*:
- Просматривает изменения
- Принимает или отклоняет правки

**Результат**:
- План обновлен только после решения стройконтроля
