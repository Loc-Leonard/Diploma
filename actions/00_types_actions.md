# Общие правила для всех действий
- Фиксация времени и гео при внесении данных (работы, материалы, замечания, нарушения)
- Офлайн-режим: пользователь может создать записи без связи, а система должна принять их позже без дублей
- Доступ к объекту определяется ролью + назначением на объект
