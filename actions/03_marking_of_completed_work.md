# Отметка выполненных работ (прораб) и верификация (стройконтроль)
*Цель*: ежедневная фиксация факта выполнения и последующая проверка заказчиком

*Прораб*:
1. В конце рабочего дня открывает объект -> выбирает работу (WorkItem)
2. Создает запись "выполнено":
	- объем/комментарий
	- прикладывает фото
3. Система фиксирует время + гео внесения данных
4. Статус записи: "На верификации"

*Стройконтроль*:
1. При посещении объекта открывает список записей "на верификации"
2. По каждой записи принимает решение:
	- подтвердить выполненные работы
	- отклонить (причина/комментарий)
3. Система фиксирует время + гео действия верификации

**Результат**:
- Факт работ становится "подтвержден" или возвращается на доработку
